Changelog

Version Notes Date
v0.0.6pre Added a main menu and options menu. The options menu works but there is nothing for it to change yet. Large slow ships now drop mines that take one life but are really slow. Updated the ability to create power ups. Added Shield drop item that will block one life. Player firing now upgrades from 1 shot to 5 shots after reaching 2000,4000,7000,11000,and 16000 score. 6/23/2018
v0.0.5pre Simple Ship slowed down 33%. Drop items sped up 40%. Large slower enemies added, they take 8 shots to kill and are worth 150 points. Enemies now fly in formations. More variability to asteroid spawns added. Refined code for enemy ships. 6/22/2018
v0.0.4pre Created framework for drop items. Added health pack which is dropped by asteroids 25% of the time. Added coin that drops from shot down ships 100% of the time. It is worth 15 points. 6/22/2018
v0.0.3pre Gave player 3 lives and a UI to keep track of score, wave number, and lives. Asteroids are worth 30 points and Ships are worth 50 points. Gave asteroid a temporary model. Refined input so it didn't clutter start setup. Added simple particles. Fixed game aspect ratio to 16:9. Added ability to exit game with escape. 6/22/2018
v0.0.2pre Game ends when player is hit by anything. Enemies now come in waves. 6/21/2018
